Project Title:

    Sub Trakcr

Features:

    - Google OAUTH Login
    - Add, Edit, Delete Subscriptions
    - History of the previous subscriptions by platforms
    - Flexible Billing Cycles (Weekly/Monthly/Yearly/Custom)
    - Email Remainders before renenwals
    - Interactive Dashboard Charts or PDF
    - Light / Dark Theme
    - Mobile-first Responsive UI
    - Server Actions-Frist Architecture
    - PostgreSQL + Prisma ORM
    - Clean, Scalable

Tech Stack:

    Frontend:

        - Next.js (App Router)
        - TypeScript
        - Tailwind CSS
        - Shadcn UI
        - Apache Echarts
        - Zustand
        - Framer motion

    Backend:

        - Next.js Server Actions
        - Next.js API Routes (for auth, webhooks, exports)
        - PostgreSQL
        - Prisma ORM
        - Better AUTH
        - Resend

    Deployment:

        - Vercel
